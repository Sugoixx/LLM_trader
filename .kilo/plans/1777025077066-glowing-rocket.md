# Plan: Avoid SELL signals in uptrends

## Context
User wants to avoid selling (SELL signals) during bullish/uptrends ("en trend haussiere il faut eviter de sell...possible mais pas souvent"). Currently, the Bollinger Reversion strategy can generate SELL signals in uptrends with 0.50 confidence.

## Current Behavior
- **Bollinger Reversion** (`bollinger_reversion.py:103-112`): Sells on upper band touch with 0.70 confidence in downtrend, 0.50 in uptrend
- **RSI Crossover** (`rsi_crossover.py:105-116`): Only sells in downtrend (GOOD - no change needed)
- **Fast Trader** (`fast_trader.py:41`): Filters signals below 0.55 confidence, so Bollinger SELL in uptrend (0.50) gets filtered - but only when using Fast Trading Mode
- **LLM signals** (`trading_strategy.py:820`): No trend filter for SELL signals

## Changes

### 1. Modify `bollinger_reversion.py` - Block SELL in uptrends
**File**: `src/trading/algo_strategies/bollinger_reversion.py`  
**Lines**: 103-127

**Change**: Only generate SELL signals when `is_downtrend` is True. In uptrend, return HOLD instead of SELL.

Current logic (lines 92-112):
```python
if prev_high >= upper[-1]:
    if not vol_confirmed:
        return AlgoSignal(...)  # HOLD - low volume
    return AlgoSignal(
        signal="SELL",
        confidence=0.70 if is_downtrend else 0.50,  # Sells in uptrend with 0.50!
    )
```

New logic:
```python
if prev_high >= upper[-1]:
    if not vol_confirmed:
        return AlgoSignal(...)  # HOLD - low volume
    if not is_downtrend:
        # In uptrend, avoid selling - return HOLD
        return AlgoSignal(
            strategy_name=self.name,
            signal="HOLD",
            explanation=(
                f"Price touched upper band ({upper[-1]:.4f}) at {prev_high:.4f} "
                f"but trend is UP (price {price:.4f} > SMA{self.trend_period} {sma_val:.4f}). "
                f"Avoiding sell in uptrend."
            ),
            confidence=0.30,
        )
    # Only sell in downtrend
    return AlgoSignal(
        signal="SELL",
        confidence=0.70,  # Higher confidence since trend-aligned
        ...
    )
```

### 2. (Optional) Add trend filter for LLM signals in `trading_strategy.py`
**File**: `src/trading/trading_strategy.py`

# After line 749 (market_conditions extracted)
# Block SELL signals in strong uptrends
if signal == "SELL":
    trend_dir = market_conditions.get("trend_direction", "NEUTRAL")
    if trend_dir == "BULLISH":

            "[SELL-BLOCKED] SELL signal in BULLISH trend (%s) — avoiding sell in uptrend.",
